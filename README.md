# 🎯 Number Guessing Game

The **Number Guessing Game** is a fun and interactive browser-based game where the player tries to guess a randomly generated number between **1 and 100**. The game provides hints after every guess and allows up to **10 attempts**.

## 🕹️ Live Demo

🔗 [Play Now](https://drishti-cs1251.github.io/Number-Guessing-Game/)

## ✨ Features

- 🎲 Random number generation between 1 and 100
- ⌨️ User input with validation
- 🔁 Real-time feedback:
  - "Too high!"
  - "Too low!"
  - "Yay! You guessed it right..."
- 🧠 Keeps track of previous guesses
- ❌ Displays remaining attempts
- 🔄 Option to **Start a New Game** after win or game over

## 💻 Technologies Used

- **HTML** – Structure of the game interface
- **CSS** – Colorful styling with gradients and layouts
- **JavaScript** – Game logic, random number generation, input handling

## 🎮 How to Play

1. The game randomly picks a number between **1–100**.
2. Enter your guess in the input box.
3. Click the **Submit Guess** button.
4. You'll receive a hint:
   - 🔺 “Number is too high...”
   - 🔻 “Number is too low...”
   - ✅ “Yay! You guessed it right...”
5. You have **10 chances** to guess the number.
6. If you fail to guess, the game ends and shows the correct number.
7. Click **Start New Game** to try again.
